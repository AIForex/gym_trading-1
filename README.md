# OpenAI Gym for Trading
An example of Reinforcement Trading using Q-learning Algorithm

Requirements:
-Pandas
-Matplotlib
-Numpy
-gym
-TA-Lib
-baselines
-tensorflow

This framework was built in Python 3.5.2
